# 회사 경비 처리 규정 및 가이드

## 1. 경비 처리 개요

본 문서는 (주)테스트컴퍼니의 경비 처리 규정을 안내합니다. 모든 임직원은 업무 관련 경비 발생 시 본 규정에 따라 처리해야 합니다.

## 2. 경비 항목 분류

### 2.1 교통비
- **택시비**: 건당 최대 50,000원, 야간(22시 이후) 귀가 시 한도 없음
- **대중교통**: 실비 정산, 영수증 필수
- **출장 교통비**: KTX/항공 이코노미 기준, 부장급 이상 비즈니스 허용
- **자차 운행**: km당 300원 정산 (주유 영수증 불필요)

### 2.2 식대
- **업무 식대**: 1인당 15,000원 이내 (점심), 30,000원 이내 (저녁)
- **접대비**: 1인당 50,000원 이내, 사전 팀장 승인 필요
- **야근 식대**: 20시 이후 근무 시 15,000원 한도

### 2.3 숙박비
- **국내 출장**: 1박당 최대 150,000원 (서울/수도권), 120,000원 (지방)
- **해외 출장**: 국가별 상한액 별도 적용 (해외출장규정 참조)

### 2.4 업무용품 구매
- **소모품**: 건당 100,000원 이하 팀장 승인
- **장비/기기**: 건당 500,000원 이상 부서장 승인 필요
- **소프트웨어**: IT팀 사전 검토 후 구매

### 2.5 교육비
- **외부 교육**: 건당 500,000원 이내 팀장 승인, 초과 시 부서장 승인
- **도서 구입비**: 월 50,000원 한도

## 3. 경비 처리 절차

### 3.1 신청 절차
1. 경비 발생일로부터 **7영업일 이내** 신청
2. ERP 시스템 > 경비관리 > 경비신청 메뉴에서 입력
3. 증빙자료(영수증, 카드전표 등) 첨부 필수
4. 결재라인: 신청자 → 팀장 → 부서장 (100만원 초과 시 경영지원팀장 추가)

### 3.2 증빙 서류
- **법인카드 사용**: 카드 전표 자동 연동
- **개인카드/현금**: 영수증 원본 스캔 첨부
- **간이영수증**: 건당 30,000원 이하만 인정
- **전자 영수증**: 원본 파일(PDF) 첨부

### 3.3 정산 일정
- 매월 **1일~10일**: 전월 경비 신청 마감
- 매월 **15일**: 경영지원팀 검토 완료
- 매월 **25일**: 급여와 함께 정산 지급

## 4. 법인카드 사용 규정

### 4.1 발급 기준
- 팀장급 이상: 개인 법인카드 발급
- 일반 직원: 팀 공용 법인카드 사용

### 4.2 사용 한도
| 직급 | 월 한도 | 건당 한도 |
|------|---------|-----------|
| 사원/대리 | 500,000원 | 200,000원 |
| 과장/차장 | 1,000,000원 | 500,000원 |
| 부장 | 2,000,000원 | 1,000,000원 |
| 임원 | 5,000,000원 | 별도 협의 |

### 4.3 사용 제한
- 개인 용도 사용 절대 금지
- 현금서비스/카드론 사용 금지
- 유흥업소 사용 금지
- 상품권/기프트카드 구매 금지

## 5. 출장 경비 규정

### 5.1 국내 출장
- **일비**: 20,000원/일 (직급 무관)
- **숙박비**: 항목 2.3 참조
- **식비**: 1일 50,000원 한도

### 5.2 해외 출장
- **항공**: 이코노미 기준 (8시간 초과 비행 시 비즈니스 허용 - 임원급)
- **일비**: 국가별 상이 (미국 $80, 일본 ¥8,000, 유럽 €70)
- **숙박비**: 국가별 상한액 적용 (미국 $200, 일본 ¥20,000, 유럽 €180)

### 5.3 출장 보고
- 출장 완료 후 **3영업일 이내** 출장보고서 제출
- 정산 시 출장보고서 첨부 필수

## 6. 자주 묻는 질문 (FAQ)

**Q1. 영수증을 분실했을 경우 어떻게 하나요?**
A: 영수증 분실 사유서를 작성하고 팀장 확인 후 제출합니다. 단, 건당 50,000원 이상은 원본 영수증 필수이므로 재발급을 받아야 합니다.

**Q2. 경비 신청 기한(7영업일)을 초과했을 경우?**
A: 팀장 승인 후 경영지원팀에 사유서를 제출하면 예외 처리 가능합니다. 단, 분기 내 경비만 소급 처리 가능합니다.

**Q3. 법인카드 한도 초과 시 개인카드로 결제해도 되나요?**
A: 가능합니다. 개인카드 사용 시 영수증과 함께 개인카드 사용 사유서를 제출하면 정산됩니다.

**Q4. 접대비 사전 승인을 못 받았을 경우?**
A: 긴급 상황 시 사후 승인이 가능하나, 반드시 24시간 이내 팀장에게 구두 보고 후 3영업일 이내 서면 승인을 받아야 합니다.

**Q5. 해외 출장 시 환율은 어떻게 적용하나요?**
A: 카드 결제 시 카드사 매입 환율 적용, 현금 사용 시 출장일 기준 서울외국환중개 매매기준율을 적용합니다.

## 7. 경비 처리 위반 시 제재

- **1차 위반**: 서면 경고 및 해당 경비 환수
- **2차 위반**: 법인카드 사용 정지 (3개월)
- **3차 위반**: 인사위원회 회부 및 징계 조치
- **허위 청구**: 즉시 징계 조치 및 해당 금액 3배 환수
